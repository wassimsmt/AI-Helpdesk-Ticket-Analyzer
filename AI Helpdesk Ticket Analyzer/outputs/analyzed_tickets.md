* **Ticket 1**



Category: Network / Connectivity

Priority: Medium

Likely cause: Possible unstable Wi-Fi signal, network adapter driver issue, or router instability

Troubleshooting steps:



Confirm if other devices experience the same disconnections

Restart router/modem

Forget and reconnect to the Wi-Fi network

Update or reinstall Wi-Fi drivers

Run OS network troubleshooter

Test with another network to isolate device vs network issue

Escalate if: Issue persists after driver and router checks, or impacts multiple users



* **Ticket 2**



Category: Email / Application (Outlook)

Priority: High

Likely cause: Possible mail server connectivity issue or incorrect Outlook configuration

Troubleshooting steps:



Verify internet connectivity

Ensure Outlook is not in offline mode

Restart Outlook and the computer

Check account settings (server, ports, credentials)

Test email access via webmail

Temporarily disable VPN/firewall for testing

Escalate if: Issue occurs for multiple users or persists despite correct configuration



* **Ticket 3**



Category: System Performance

Priority: High

Likely cause: Possible compatibility or resource issue after Windows update

Troubleshooting steps:



Restart the system

Check Task Manager for high CPU, RAM, or disk usage

Disable unnecessary startup programs

Check for pending updates or patches

Roll back recent Windows update if issue started after update

Repair or reinstall Excel

Escalate if: System continues freezing after rollback or affects multiple devices



* **Ticket 4**



Category: Account / Access

Priority: High

Likely cause: Forgotten password or possible account lock

Troubleshooting steps:



Use password reset/self-service portal

Verify identity using available methods (MFA/security questions)

Confirm correct username is used

Check if account is locked

Escalate if: User cannot reset password or account remains locked



* **Ticket 5**



Category: Printing

Priority: Medium

Likely cause: Possible stuck print queue or print spooler issue

Troubleshooting steps:



Cancel all pending print jobs

Restart print spooler service

Restart printer and computer

Verify printer connection and status

Update or reinstall printer drivers

Escalate if: Print queue remains stuck or issue affects multiple users/printers
